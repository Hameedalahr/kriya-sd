# YOLOv8 Person Detection in Video

## Description
This project detects persons in a given video using YOLOv8.
It draws bounding boxes, saves frames when a person appears,
and generates an output video with detections.

## Tools Used
- Python
- OpenCV
- YOLOv8 (Ultralytics)

## How to Run
1. Install dependencies:
   pip install ultralytics opencv-python
2. Place input video in:
   data/video/raw/
3. Run:
   python main.py

## Output
- Output video with detected persons
- Frames saved when a person appears
